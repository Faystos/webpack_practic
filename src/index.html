Какойто заголовок

В данном случае не используется имя для импортированных данных, а также отсутствует ключевое слово from. При импорте не js файлов необходимо указывать расширение импортируемого файла.

Если файл с модулем содержит экспортируемое значение по умолчанию, то для импорта такого значения фигурные скобки не нужны и переменная, в которую записывается экспортируемый код может иметь любое валидное имя. default может быть только один.
